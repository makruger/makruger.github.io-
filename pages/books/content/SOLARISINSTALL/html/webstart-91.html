Chapter 2

Installing With the Solaris Installation Program (Tasks)

This chapter explains how to use the Solaris installation program on the Solaris Operating System DVD or Solaris Software - 1 CD to install or upgrade Solaris software.

This chapter contains the following topics:

  • SPARC: Performing an Installation or Upgrade With the Solaris Installation Program

  • x86: Performing an Installation or Upgrade With the Solaris Installation Program

For information about new installation features, see Chapter 2, What’s New in Solaris Installation, in Solaris Express Installation Guide: Planning for Installation and Upgrade. For information about all the new features in the Solaris OS, see Solaris Express, Developer Edition What’s New.
