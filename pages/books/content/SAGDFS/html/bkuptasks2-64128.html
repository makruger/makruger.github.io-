Chapter 25

Backing Up Files and File Systems (Tasks)

This chapter describes the procedures for backing up file systems by using the ufsdump command.

For information on these procedures, see Backing Up Files and File System (Task Map).

For overview information about performing backups, see Chapter 24, Backing Up and Restoring File Systems (Overview).

For information about backing up individual files to diskettes, see Chapter 29, Copying UFS Files and File Systems (Tasks).

For additional information on the ufsdump command, see Chapter 28, UFS Backup and Restore Commands (Reference).
