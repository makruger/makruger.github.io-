Chapter 1

Introduction to the Distribution Constructor

The distribution constructor is a tool that application developers can use to build their own custom OpenSolarisTM image which they can then distribute to their contacts and customers.

  • If you have not used the distribution constructor before, read this introduction. Then, install the tool on your system, using the instructions at Chapter 2, Setup for the Distribution Constructor.

  • If you are already familiar with the distribution constructor tool, go to Chapter 3, Using the Distribution Constructor for detailed instructions about how to design and create images.
