4. Contributing

  We welcome contributions to these documents!

  You'll need to sign a standard OpenSolaris contributor agreement
  first.  Next, send a your contributor agreement # and a pointer to
  your changes to docs-discuss@opensolaris.org with a request for
  review.  After review by the community is complete, one of the
  docs community sponsors will integrate your changes or 
  help you set up a new Project and repository for the document.
