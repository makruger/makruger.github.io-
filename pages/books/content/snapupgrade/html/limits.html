Limitations in Current Release

Note the following limitations in the OpenSolaris 2009.06 release.

  • The installation of the OpenSolaris OS supports one pool on one disk. After installation, you can attach another disk to the root pool and set up another pool.

  • The beadm utility is implemented with ZFS support only.

  • The beadm utility manages boot environments and snapshots that were created by the beadm utility. If a user manually creates the datasets for a boot environment by using the zfs(1M) command, the beadm utility should recognize the boot environment, but management of this boot environment might be undefined.
