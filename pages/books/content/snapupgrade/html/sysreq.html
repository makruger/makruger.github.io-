beadm Utility Requirements

To use the new beadm utility to manage your boot environments, your system must meet the following requirements.

Table 1-1 System Requirements

Requirement

Description

OpenSolaris 2008.05 or later release

You must have the OpenSolaris 2008.05 release or a later release installed on your system. Prior OpenSolaris releases did not include the beadm utility. For information about installing the OpenSolaris operating system, see Getting Started With OpenSolaris.

ZFS file systems

All OpenSolaris Developer releases have ZFS file systems. The beadm utility is designed for ZFS file systems.
