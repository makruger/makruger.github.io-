System Requirements

Before installing the Developer release, verify that your system meets all of the requirements described in the Solaris Express Developer Edition System Requirements.

The installation requires a minimum of 768 Mbytes memory.

SPARC Developer System

This release of Solaris Express Developer Edition is only for x86 platforms. However, developers on SPARC systems can obtain similar functionality by downloading Solaris Express Community Edition and then installing Sun Studio and NetBeans IDE 5.5 with NetBeans Enterprise Pack 5.5.

Downloads are available at the following Web sites:

  • Solaris Express Community Edition

  • Sun Studio Express June 2007

  • NetBeans IDE 5.5.

    Note - Click on the NetBeans 5.5 Download button. Then, you can either proceed with downloading that tool, or select Cancel and download one of the following products:

    • NetBeans Enterprise Pack 5.5

    • NetBeans Visual Web Pack 5.5

    • NetBeans Profiler 5.5
