Executing Privileged Commands

The OpenSolaris operating system exploys the pfexec utility to execute commands with predefined attributes that the system administer specifies in user profiles. The system administrator assigns privileges to groups of users in a user profile. These privileges can restrict or permit file access for the group, restrict or permit the group's ability to perform various tasks, and so on.

See the pfexec man page for complete information about using this utility.
