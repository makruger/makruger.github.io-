Installation Roadmap

You have a number of options for installing the current OpenSolaris release from the OpenSolaris Live CD to your system.

  1. Choose one of the following options:

    • To install the OpenSolaris 2009.06 release from the Live CD onto your system, continue to Steps 2 and 3.

    • If you do not have the OpenSolaris 2009.06 Live CD, go to the http://www.opensolaris.com/get/ site and select a link to download the image for the current OpenSolaris release to your system. Once you have downloaded the image, choose one of the following options:

      • Burn this image to a CD and continue to Steps 2 and 3.

      • Save the image on your system and mount it in a virtual machine as described in Running OpenSolaris in VirtualBox.

    • To install or mount the OpenSolaris release from the Live CD into a virtual machine, see Virtualization Options.

    • If you already have an OpenSolaris release installed on your system, and you want to upgrade to the OpenSolaris 2009.06 release, do not use these installation instructions. Instead, go to Upgrading to the Next OpenSolaris Release.

  2. Check the requirements and limitations for running the installer on your x86 based system:

    1. Verify that your x86 based system meets the System Requirements.

    2. Review the Installation Restrictions.

    3. Verify that you have installed all necessary device drivers by using the Device Driver Utility 1.1. See Managing Device Drivers on Your System.

  3. Choose one of the following alternatives for installing from the Live CD on your system:

    • If the OpenSolaris operating system will be the only operating system installed on your x86 based system, see instructions for Installing OpenSolaris 2009.06 From the Live CD.

    • If you will be setting up a multiboot environment, review the specifications in Preparing for a Multiboot Environment.

      1. Back up your system. This step is strongly recommended when repartitioning your system. See the backup instructions in step one for Partitioning a Multiboot System.

      2. Continue with the steps for Partitioning a Multiboot System. If you already have a Solaris fdisk partition on your system, the installation overwrites this partition.

      3. After you have partitioned your system, see the instructions to Installing OpenSolaris 2009.06 From the Live CD.

For installation issues, see Chapter 9, Appendix A: Troubleshooting.
