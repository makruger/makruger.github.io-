What is OpenSolaris?

OpenSolaris has several meanings, depending on the context in which it is used.

For example, OpenSolaris is:

  • An operating system code base originally based on Sun's proprietary SolarisTM Operating System.

  • An open source development project.

  • A free binary distribution of the OpenSolaris software that is available for redistribution under the terms of the Common Development and Distribution License (CDDL).

This guide is intended to help you install and set up the free binary distribution of the OpenSolaris software and make your initial experience with it a successful one.

For more information, see Related Information.
