Package Categories

The package categories are displayed in the left panel of the Package Manager. See the Package Manager (GUI) for information about the Package Manager.

Note - A package subcategory is displayed only if there are packages available in that subcategory.

If you are using the command—line interface (CLI), the category of a package is shown by the package metadata, info.classification.

If you are creating and publishing an IPS package, you might use the info.classification attribute with the publication client, pkgsend(1), to categorize your package before publishing the package. See the Actions in IPS section to learn about actions and their corresponding attributes of a package.
