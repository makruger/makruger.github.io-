Setting Up Your OpenSolaris Development Environment

The OpenSolaris 2009.06 release offers complete support for developing and deploying applications for the web. The software needed for development of web applications are available as packages in the http://pkg.opensolaris.org/dev and http://pkg.opensolaris.org/release repositories.

See Setting up Your OpenSolaris Development Environment to learn how to download and install applications such as Sun Studio compilers and NetBeansTM software and to obtain the tools required to set up a Web Stack Development environment.
