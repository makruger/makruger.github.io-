Chapter 9

Experimental Java Bindings for the Solaris Trusted Extensions Label APIs

This chapter describes an experimental set of JavaTM classes and methods that mirror the label application programming interfaces (APIs) that are provided with the Solaris Trusted Extensions software. The Java implementation of the Trusted Extensions label APIs is intended to be used to create label-aware applications. As a result, all of the label APIs provided by Trusted Extensions are not part of the Java implementation.

The presentation of these experimental Java APIs (Java bindings) demonstrate how the Solaris Trusted Extensions features can be expanded into the Java development environment.

Caution - These experimental Java bindings are not a supported part of the Solaris Trusted Extensions software.
