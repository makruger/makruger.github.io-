Chapter 5

How to Administer a Custom Install Client

You can install the OpenSolaris OS to multiple client systems by using your install server. If all your client systems have the same hardware characteristics, then the default installation specifications that you setup by using the installadm create-service command can be used for all client installations. However, client setup is required if you want to install the OpenSolaris OS to clients with varying hardware characteristics. For example, if you want to install to both x86 clients and to SPARC clients, client setup is required.

Note - If you plan to install to SPARC client systems, those systems must support WAN boot. For more information, see 2. Does my SPARC Client Support WAN boot?
