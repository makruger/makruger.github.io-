How to Delete an Install Client

Use the following command to delete an existing install client.

# installadm delete-client macaddr

This command deletes an existing client's specific service information that was previously set up using the installadm create-client command.

The macaddr field is required to specify the MAC address for the install client.

For help with client installations, see Chapter 7, How to Manage Client Installations and 2. What if the Client Installation Fails?
