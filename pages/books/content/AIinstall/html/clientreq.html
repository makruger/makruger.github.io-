Client System Requirements

The client systems for automated installation must meet the following requirements:

Note - Any system that meets these requirements can be used as an automated install client, including laptops, desktops, virtual machines, and enterprise servers.

Table 7-1 Client Requirements

Platform

Requirement

SPARC Clients

  • Supports WAN Boot in the Open Boot Prompt

    See 2. Does my SPARC Client Support WAN boot?

  • Minimum memory: 1 GB

  • Minimum disk space: 12 GB

x86 Clients

  • Minimum memory: 512 MB

  • Minimum disk space: 12 GB
